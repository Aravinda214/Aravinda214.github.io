Applied AI web application that combines multiple ML components to support structured mental-health and relationship-support workflows.

<!--more-->

## Highlights

- Combined prediction, emotional analysis, peer clustering, and route optimization in one workflow.
- Built the application as a Flask-based end-to-end system rather than a single model demo.
- Used NLP and graph-style optimization techniques to support different parts of the user journey.
- Focused on an accessibility-oriented, socially meaningful use case.

## Tech Stack

Flask, Python, HTML, CSS, JavaScript, machine learning, NLP, and graph algorithms.

## Why It Matters

This project shows full-stack applied ML thinking across prediction, language processing, workflow design, and user-facing delivery.
